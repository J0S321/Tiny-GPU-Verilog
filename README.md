# Tiny-GPU-Verilog
Learning GPU architecture by building a small Verilog-based GPU for parallel matrix operations. 
#Inspiration: 
This project was inspired by Adam Maj's Tiny GPU project which is a tiny GPU implementaiton using Verilog design to learn GPU architecture fromt he ground up. My goal with this repository is to ahve my own understanding of GPU architecture, Verilog design, and parallel computation by implementing and documenting similar concepts step by step
