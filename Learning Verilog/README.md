# Learning Verilog

The initial step I took for this project was learning Verilog. 

Although I have prior experience with VHDL from ESE 382, I chose to use Verilog for this project because of its C-like syntax and because it's commonly used in digital design projects. Since I already understood the fundamentals of hardware description languages from VHDL, Verilog felt like the next step for building and simulating hardware modules better. 

This folder contains some early design practice modules, including basic combinationa nd sequential circuits. These modules helped me build the foundation needed for the larger goal of understanding GPU architecture and eventually making a GPU design in Verilog
